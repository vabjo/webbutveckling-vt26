Arbetsprocess

Upplägg

  • Vite installeras med npm install --save-dev vite
  • Nytt projekt skapas med npm create@latest, där man namnger projektet, anger framework och väljer variant
  • Plugin installeras: Vite Image Optimizer med npm install vite-plugin-image-optimizer --save-dev, och Sharp med npm install sharp --save-dev
  • För att kunna exportera fler sidor än index.html med Vite läggs sidorna index, photos och process till i vite.config.js
  • I den filen konfigueras även vite image optimizer, så att bilderna komprimeras till en mindre filstorlek

Programmering

  • Projektet struktureras med HTML: sidorna index.html, photos.html och process.html skapas. Alla sidorna består av ett sidhuvud, huvudinnehåll och en sidfot
  • CSS används för att formge projektet: huvudsakligen används display: flex för att placera element, men även display: grid används för att skapa en layout för huvudinnehållet på sidorna Bilder och Process
  • För responsivitet anges brytpunkterna min-width: 480px och min-width: 768px
  • Ett litet program skrevs i JavaScript som visar en bild vid ett knapptryck på indexsidan

Publicering

  • Github används för versionhantering, och två grenar skapades: main för publicering och dev för utveckling
  • Vid publicering används Vite för att minifiera och optimera CSS och JavaScript koderna, samt skapandet av en mapp som innehåller de optimerade filerna som används för publicering
  • Vite optimerar även webbplatsens bilder genom Vite Image Optimizer pluginen, och placerar de förminskade filerna i publiceringsmappen
  • Slutligen publiceras webbplatsen genom Github pages från källförrådets huvudgren
